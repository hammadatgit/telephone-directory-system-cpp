# Telephone Directory System (C++)

## Overview

The Telephone Directory System is a console-based application developed in C++ that allows users to store, manage, and retrieve contact information efficiently. The project demonstrates the use of file handling, data management, and menu-driven programming techniques to simulate a simple contact directory.

This project was developed as part of academic coursework to strengthen programming fundamentals and practical software development skills.

---

## Features

* Add new contacts
* View stored contacts
* Search contacts
* Update contact information
* Delete contact records
* Store data using files for persistence
* Interactive menu-driven interface

---

## Technologies Used

* C++
* Standard Template Library (STL)
* File Handling (fstream)
* Object-Oriented Programming Concepts
